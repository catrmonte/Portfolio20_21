About

This website is to showcase the projects I have completed in Game Engine Lab (CS434) during the spring 2020 semester. This course followed two text books, Hands-On Game Development Patterns with Unity by David Baron and Game Programming Patterns by Robert Nystrom. The books will be refered to as Baron and Nystrom. The two development environments used are Unity and Unreal. Assume Unity is used unless stated otherwise. There are 8 different projects used in GitHub. 7 of the 8 you can play the project online, 1 will be a video demo. You can visit the source code all eight as well. Links to both will be in the project buttons below.

I am a software engineering student, and this was my first class that is realted to the Games and Sims department. The beginning of the class was very difficult for me, but as time pressed on, I understood the consepts better and it refects in my work. I am proud with what I have accomplished in this course, and I am excited to put these skills into practice.

Project 1

Command Log

This project covered following the basics of unity following this tutorial. After that was complete, a replay functionality was added by implementing a command log and a scoring functionality by implementing the observer pattern. The command log and observer patter can be found in Baron Ch 7-9 and Nystrom Ch 2&4. You can play the game here. You can also see the source code here.

Project 2

Observer Pattern

This project builds off of the Project 1. It is recreated to allow multi-directional movement. This project implements finite state machines to make the scoring more complex. The finite state machine can be found in Baron Ch 10-11 and Nystrom Ch 7. You can play the game here. You can also see the source code here.

Project 3

Prototype and Factory Patterns

This projects follows an example given in Baron ch 3 to implement a spawn system using prototypes, factories, and profiling. Prototypes and factories can be found in Baron Ch 3-4 and Nystrom Ch 5. The output of the game is in debug statements, so the console must be open on the browser, as shown in the picture above. You can play the game here. You can also see the source code here.
